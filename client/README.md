# React + Vite

This project is a starter template for building React applications using [Vite](https://vitejs.dev/) and [Tailwind CSS](https://tailwindcss.com/).

## Getting Started

1. **Clone the repository:**
  ```bash
  git clone https://github.com/SiddharthJiyani/vite-tailwind-starter-pack.git
  cd vite-tailwind-starter-pack
  ```

2. **Install dependencies:**
  ```bash
  npm install
  ```

3. **Start the development server:**
  ```bash
  npm run dev
  ```
